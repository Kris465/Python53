# Конспект лекции: 2.1 Введение в HTML. Структура документа

## 1. Что такое HTML?

### Определение
**HTML** (HyperText Markup Language) — язык разметки гипертекста, который используется для создания структуры веб-страниц. HTML — это не язык программирования, а язык разметки, который сообщает браузеру, как отображать контент.

### Аналогия
Представьте HTML как скелет здания:
- **HTML** = Каркас и планировка (где что находится)
- **CSS** = Дизайн и отделка (как выглядит)
- **JavaScript** = Электричество и механизмы (что делает)

## 2. Основные понятия

### Теги (Tags)
Теги — это элементы HTML, которые определяют структуру и содержание веб-страницы. Теги заключаются в угловые скобки `< >`.

```html
<!-- Примеры тегов -->
<h1>Заголовок</h1>  <!-- Тег заголовка -->
<p>Абзац текста</p> <!-- Тег параграфа -->
```

### Элементы HTML
Элемент состоит из открывающего тега, содержимого и закрывающего тега:

```html
<тег>содержимое</тег>
<!-- Пример -->
<p>Это текст внутри параграфа.</p>
```

### Атрибуты (Attributes)
Атрибуты предоставляют дополнительную информацию об элементах и размещаются внутри открывающего тега:

```html
<тег атрибут="значение">содержимое</тег>
<!-- Пример -->
<a href="https://example.com">Ссылка</a>
```

## 3. Базовая структура HTML-документа

### Обязательная структура каждого HTML-документа:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Моя первая страница</title>
</head>
<body>
    <!-- Содержимое страницы здесь -->
</body>
</html>
```

## 4. Разбор структуры документа

### 1. `<!DOCTYPE html>`
Объявляет тип документа и версию HTML (HTML5).

```html
<!DOCTYPE html>
<!-- Говорит браузеру: "Это документ HTML5" -->
```

### 2. `<html>` — корневой элемент
Контейнер для всего HTML-документа. Атрибут `lang` указывает язык страницы.

```html
<html lang="ru">
<!-- lang="ru" помогает поисковым системам и скринридерам -->
```

### 3. `<head>` — заголовок документа
Содержит метаинформацию о странице (не отображается на странице).

```html
<head>
    <meta charset="UTF-8">  <!-- Кодировка символов -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Название страницы</title>
</head>
```

#### Элементы внутри `<head>`:

**Мета-теги:**
```html
<meta charset="UTF-8">
<!-- Определяет кодировку символов (обязательно для русского языка) -->

<meta name="viewport" content="width=device-width, initial-scale=1.0">
<!-- Адаптивность для мобильных устройств -->

<meta name="description" content="Описание страницы для поисковых систем">
<meta name="keywords" content="ключевые, слова, через, запятую">
```

**Заголовок страницы:**
```html
<title>Мой сайт - Главная страница</title>
<!-- Отображается на вкладке браузера и в результатах поиска -->
```

**Подключение стилей:**
```html
<link rel="stylesheet" href="styles.css">
<!-- Подключает внешний файл CSS -->
```

**Подключение скриптов:**
```html
<script src="script.js"></script>
<!-- Подключает JavaScript -->
```

### 4. `<body>` — тело документа
Содержит всё видимое содержимое страницы.

```html
<body>
    <!-- Здесь размещается весь контент -->
    <h1>Привет, мир!</h1>
    <p>Моя первая веб-страница.</p>
</body>
```

## 5. Базовые теги для контента

### Заголовки (`<h1>` - `<h6>`)
Определяют заголовки разного уровня важности.

```html
<h1>Главный заголовок (уровень 1)</h1>
<h2>Подзаголовок (уровень 2)</h2>
<h3>Заголовок раздела (уровень 3)</h3>
<h4>Заголовок подраздела (уровень 4)</h4>
<h5>Мелкий заголовок (уровень 5)</h5>
<h6>Самый мелкий заголовок (уровень 6)</h6>
```

**Важно:** Используйте только один `<h1>` на странице (для SEO).

### Параграфы (`<p>`)
Для текстовых абзацев.

```html
<p>Это первый абзац текста.</p>
<p>Это второй абзац текста.</p>
<p>Это третий абзац текста.</p>
```

### Блочные и строчные элементы

#### Блочные элементы (Block)
Занимают всю доступную ширину, начинаются с новой строки:

```html
<div>Это блочный элемент</div>
<p>Это тоже блочный элемент</p>
<h1>И это блочный элемент</h1>
```

#### Строчные элементы (Inline)
Занимают только необходимую ширину, не начинают новую строку:

```html
<span>Это строчный элемент</span>
<a href="#">Ссылка тоже строчная</a>
<strong>И этот элемент строчный</strong>
```

### Тег `<div>` (Division)
Универсальный блочный контейнер для группировки элементов.

```html
<div class="container">
    <h2>Заголовок</h2>
    <p>Текст внутри контейнера</p>
</div>

<div id="header">Шапка сайта</div>
<div id="content">Основное содержимое</div>
<div id="footer">Подвал сайта</div>
```

### Тег `<span>` (Span)
Универсальный строчный контейнер для стилизации части текста.

```html
<p>Это обычный текст, а <span class="highlight">этот текст выделен</span>.</p>

<p>Цена: <span class="price">1000</span> рублей.</p>
```

## 6. Другие важные базовые теги

### Ссылки (`<a>`)
Создают гиперссылки на другие страницы или ресурсы.

```html
<!-- Внешняя ссылка -->
<a href="https://google.com">Перейти в Google</a>

<!-- Внутренняя ссылка -->
<a href="about.html">О нас</a>

<!-- Ссылка на email -->
<a href="mailto:email@example.com">Написать нам</a>

<!-- Ссылка с атрибутами -->
<a href="page.html" target="_blank" title="Откроется в новой вкладке">
    Открыть в новой вкладке
</a>
```

**Атрибуты:**
- `href` — адрес ссылки (обязательный)
- `target` — где открыть (`_blank` — новая вкладка)
- `title` — всплывающая подсказка

### Изображения (`<img>`)
Вставляют изображения на страницу.

```html
<img src="cat.jpg" alt="Милый котенок" width="300" height="200">
```

**Атрибуты:**
- `src` — путь к изображению (обязательный)
- `alt` — альтернативный текст (обязательный для доступности)
- `width` и `height` — размеры изображения

### Списки

#### Нумерованный список (`<ol>`)
```html
<ol>
    <li>Первый пункт</li>
    <li>Второй пункт</li>
    <li>Третий пункт</li>
</ol>
```

#### Маркированный список (`<ul>`)
```html
<ul>
    <li>Элемент списка</li>
    <li>Еще один элемент</li>
    <li>И еще один</li>
</ul>
```

#### Список определений (`<dl>`)
```html
<dl>
    <dt>HTML</dt>
    <dd>Язык разметки веб-страниц</dd>
    
    <dt>CSS</dt>
    <dd>Язык стилей для веб-страниц</dd>
</dl>
```

### Линия-разделитель (`<hr>`)
```html
<p>Текст выше линии</p>
<hr>
<p>Текст ниже линии</p>
```

### Перенос строки (`<br>`)
```html
<p>Первая строка<br>Вторая строка<br>Третья строка</p>
```

## 7. Семантические теги (HTML5)

Теги, которые описывают смысл содержимого:

```html
<header>Шапка сайта</header>
<nav>Навигационное меню</nav>
<main>Основное содержимое</main>
<section>Секция/раздел</section>
<article>Самостоятельная статья</article>
<aside>Боковая панель</aside>
<footer>Подвал сайта</footer>
```

## 8. Полный пример HTML-страницы

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Пример HTML-страницы для обучения">
    <title>Учебная HTML-страница</title>
    <style>
        body { font-family: Arial, sans-serif; margin: 20px; }
        .highlight { background-color: yellow; }
        .container { border: 1px solid #ccc; padding: 20px; }
    </style>
</head>
<body>
    <!-- Шапка -->
    <header>
        <h1>Добро пожаловать на учебный сайт</h1>
        <nav>
            <a href="#home">Главная</a> |
            <a href="#about">О нас</a> |
            <a href="#contact">Контакты</a>
        </nav>
        <hr>
    </header>

    <!-- Основное содержимое -->
    <main>
        <section id="home">
            <h2>О HTML</h2>
            <p><strong>HTML</strong> (HyperText Markup Language) — это язык разметки для создания веб-страниц.</p>
            
            <div class="container">
                <h3>Базовые элементы:</h3>
                <ul>
                    <li>Заголовки (h1-h6)</li>
                    <li>Параграфы (p)</li>
                    <li>Ссылки (a)</li>
                    <li>Изображения (img)</li>
                </ul>
            </div>
        </section>

        <section id="about">
            <h2>О нас</h2>
            <p>Мы учим веб-разработке с нуля. Наши курсы включают:</p>
            
            <ol>
                <li>HTML и CSS</li>
                <li>JavaScript</li>
                <li>React</li>
            </ol>
        </section>

        <section id="contact">
            <h2>Контакты</h2>
            <p>Свяжитесь с нами:</p>
            <p>Email: <a href="mailto:info@example.com">info@example.com</a></p>
            <p>Телефон: <span class="highlight">+7 (123) 456-78-90</span></p>
        </section>
    </main>

    <!-- Подвал -->
    <footer>
        <hr>
        <p>&copy; 2024 Учебный проект. Все права защищены.</p>
        <p><small>Создано с помощью HTML5</small></p>
    </footer>
</body>
</html>
```

## 9. Комментарии в HTML

```html

    комментарий

<div>
    <!-- Комментарий внутри элемента -->

```

- Проверяет синтаксис, закрытие тегов, правильность атрибутов

## 11. Лучшие практики

### Правильно:
```html
<!-- Используйте правильную структуру -->
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Моя страница</title>
</head>
<body>
    <!-- Семантические теги -->
    <header>...</header>
    <main>...</main>
    <footer>...</footer>
</body>
</html>
```

### Избегайте:
```html
<!-- Не пропускайте обязательные атрибуты -->
<img src="photo.jpg">  <!-- БЕЗ alt - НЕПРАВИЛЬНО -->

<!-- Не вкладывайте блочные элементы в строчные -->
<span><div>Текст</div></span>  <!-- НЕПРАВИЛЬНО -->

<!-- Не забывайте закрывать теги -->
<p>Текст без закрывающего тега  <!-- НЕПРАВИЛЬНО -->
```
